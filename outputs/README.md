# Output Directory Structure

This directory contains all output files generated by running the exercises.

## Structure

```
outputs/
├── Exercise1_Python/
│   ├── tree_6plots.png
│   ├── tree_swap.png
│   ├── ring_graph.png
│   ├── pair_graph.png
│   ├── Gp_adjacency_matrix.txt
│   ├── Gp_incidence_matrix.txt
│   └── Gp_adjacency_list.txt
├── Exercise2_Parallel_MPI/
│   └── (output files from parallel exercises)
└── Exercise3_CPP/
    └── (compiled binaries and test outputs)
```

## What the Scripts Do

### Exercise1_Python
Run `python Exercise1_Python/1_graph_properties/exercise1.py` from the project root.
This generates:
- Graph visualizations (PNG files)
- Adjacency matrix, incidence matrix, and adjacency list (TXT files)

### Exercise2_Parallel_MPI
Run with `mpirun -n <N> python Exercise2_Parallel_MPI/3_communication_ring/mpi_ring.py`

### Exercise3_CPP
Compile with `make -C Exercise3_CPP/` and run tests.
Tests will read data from Exercise1_Python outputs automatically.

## Notes

- All outputs are automatically organized by exercise
- Files are generated with relative paths, so always run scripts from the project root
- The C++ tests will automatically find the Python-generated files in the outputs directory
