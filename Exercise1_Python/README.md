# Exercise 1 - Python Graph Algorithms

## Objectifs
Implémenter et tester différents **algorithmes de graphes** en Python.

## Structure

### 📁 1_graph_properties/
**Détection de propriétés de graphes**
- Déterminer si un graphe est **cyclique**
- Vérifier si un graphe est **bipartite**
- Vérifier si un graphe est **vide** (sans arêtes)
- Vérifier si un graphe est **complet** (toutes les arêtes possibles)

Fichier: `exercise1.py`

### 📁 2_tree_structures/
**Construction et manipulation d'arbres**
- Créer une structure d'arbre de taille n avec nœuds labelisés 1 à n
- Générer **6 plots** différents avec NetworkX
- Implémenter une fonction d'**échange de positions** entre deux nœuds
- Tester avec n=10, n=20, etc.

### 📁 3_graph_visualization/
**Visualisation avancée avec NetworkX**
- Plot circulaire d'un graphe G
- Visualiser un graphe à 8 nœuds (p0-p7) en disposition circulaire
